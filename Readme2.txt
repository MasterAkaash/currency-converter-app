# Currency Converter App

## Overview
This is a simple currency converter web application built using ReactJS. It allows users to quickly and easily convert between different currencies based on real-time exchange rates.

## Features
- **User-Friendly Interface:** Intuitive and easy-to-use interface for seamless currency conversion.
- **Real-Time Data:** Fetches the latest exchange rates from Open Exchange Rates API.
- **Responsive Design:** Ensures a consistent and pleasant user experience across various devices.
- **Customizable Styling:** Easily adaptable styling to suit different preferences and themes.

## Getting Started
To run the app locally and explore its features:
1. Clone the repository: `git clone [repository_url]`
2. Install dependencies: `npm install`
3. Obtain an API key from [Open Exchange Rates](https://openexchangerates.org/signup) and replace 'YOUR_API_KEY' in `src/App.js` with the actual key.
4. Start the development server: `npm start`
